# odin-recipes
The Odin Project ; Recipes project

A simple recipe website built with basic HTML as part of The Odin Project Foundations course.

## Project Overview

This project demonstrates:

- Basic HTML structure
- Working with links and directories
- Creating multiple web pages
- Using lists, images, headings, and paragraphs
- Building a simple multi-page website

## Features

- Homepage with links to recipes
- Multiple recipe pages
- Recipe descriptions
- Ingredients lists
- Step-by-step cooking instructions
- Navigation back to the homepage
